# Shadow Security Telegram Bot

## Project Overview
A Telegram bot built with aiogram for the Shadow Security project.

## Current Setup
- **Language**: Python 3.11
- **Framework**: aiogram 3.3.0
- **Status**: Basic bot structure created

## How to Use
1. Add your Telegram bot token to secrets (`BOT_TOKEN`)
2. The bot starts automatically with the workflow
3. Send `/start` to the bot to test
